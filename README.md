# 简介

Lite Notepad 是一款轻量的个人笔记应用。采用网页三剑客的技术，即 html、css 和 JavaScript 编写前端界面，通过 electron 技术打包。本软件支持三大系统平台，即 windows、linux 和 macOS。本软件支持云端存储功能以及离线保存，精美的 UI 界面，简单易懂的操作，让用户感受到简单易用的感觉。
